Gesture-Driven OS Controller

January - April 2025

This is a Python-based human-computer interaction (HCI) application that enables touchless control of your operating system using real-time hand gestures. Leveraging OpenCV and MediaPipe for computer vision, the system tracks hand landmarks with a standard webcam and maps specific finger movements to desktop commands. Users can glide the mouse cursor, trigger clicks with a pinch gesture, control system volume with a fist, and switch browser tabs with a rapid swipe. Designed with dynamic smoothing and safety fail-safes, this project offers a seamless, futuristic, and accessible interface for navigating your workspace without a mouse or keyboard.
